# 📌 Simulador de Concurrencia con Flask

## 🔹 Descripcion

Este proyecto consiste en la implementacion de problemas clasicos de concurrencia utilizando hilos en Python, integrados en un backend con Flask para visualizar los resultados en una interfaz web.

Se aplican mecanismos de sincronizacion para controlar el acceso a recursos compartidos y evitar condiciones de carrera.

---

## 🔹 Temas aplicados

- Mutex (Lock)
- Semaforos
- Variables de condicion (wait / notify)
- Productor - Consumidor
- Lectores - Escritores
- Barrera de sincronizacion
- Programacion concurrente con threading

---

## 🔹 Requisitos

- Python 3.x
- pip
